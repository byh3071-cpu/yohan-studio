지금까지는 캡처를 만들고, 복사하고, AI에 붙여넣었다. 이제 AI 코딩 도구가 캡처를 직접 불러온다.

- **원격 MCP 서버** — Claude Code·Cursor에서 주소 한 줄이면 연결 끝. 별도 설치 없음.
- **MCP 도구 3종** — 캡처 목록 조회(`snap_history`), Context Pack 가져오기(`snap_pack`), 분석용 요약(`snap_analyze`).
- **수집 파이프라인** — 공유 링크로 올린 캡처만 목록에 잡힌다. 자동 전송은 없다.
- **보안** — 접속 토큰 인증 + 출처 검증. 내 캡처는 나만 조회할 수 있다.

확장 프로그램 쪽 코드는 바뀐 게 없다. 브라우저에서 쓰던 그대로, 연결 통로만 하나 늘었다.

- **익명 공유 링크** — 캡처를 올리면 로그인 없이 공유 링크가 생긴다. 7일이 지나면 링크와 서버 데이터가 함께 사라진다.
- **디자인 전면 개편** — 스위스 에디토리얼 스타일로 사이드패널·오버레이를 새로 입혔다. 그라데이션을 걷어내고 아이콘과 문구를 정리했다.
- **누출 차단** — 페이지 정보 수집은 기본 꺼짐. 공유 전 최초 1회 동의를 받고, 정해진 항목만 전송한다.

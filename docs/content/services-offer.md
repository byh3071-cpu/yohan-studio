# Services 페이지 카피 SSOT (013-P2)

> Track A가 `src/data/services.ts` 작성 시 이 문서를 단일 출처로 사용.

## 공통

- **문의 CTA:** Google Form / Typeform — env `NEXT_PUBLIC_CONTACT_FORM_URL`
- **결제:** Phase 2에서는 링크만 (Stripe Phase 3)
- **톤:** 1인 기업가 대상, 과장 없음, 제공 범위 명확

---

## 1. AI'm Scan Report

| 항목 | 내용 |
|------|------|
| slug | `scan-report` |
| 가격 | 49,000 ~ 99,000원 |
| 기간 | 3~5영업일 |
| 대상 | Lab/진단을 풀었지만 **뭘 먼저 손대야 할지** 막힌 1인 기업가 |
| tagline | 21문항 진단 결과를 깊이 분석한 1:1 리포트 |

**제공물**

- 영역별 진단 결과 분석 PDF (8~12p)
- 약한 영역 TOP 3 + 90일 실행 로드맵 1장
- 30분 비동기 Q&A (영상 또는 문서 답변)

**제외**

- 직접 구현 대행, Notion 세팅 대행 (→ OS Build)

## 2. AI'm OS Template

| 항목 | 내용 |
|------|------|
| slug | `os-template` |
| 가격 | 19,000 ~ 49,000원 |
| 기간 | 즉시 다운로드 (Notion 복제) |
| 대상 | Notion으로 1인 OS 뼈대를 **빠르게** 깔고 싶은 사람 |
| tagline | Decision · Resource · Weekly Review가 묶인 Notion 템플릿 |

**제공물**

- Notion 템플릿 복제 링크
- 온보딩 가이드 PDF (5p)
- 추천 주간 루틴 체크리스트

**제외**

- 커스텀 DB 설계, 1:1 세팅 (→ OS Build)

## 3. AI'm OS Build

| 항목 | 내용 |
|------|------|
| slug | `os-build` |
| 가격 | 290,000 ~ 590,000원 |
| 기간 | 2~4주 |
| 대상 | 템플릿으로 부족하고 **내 업무에 맞게** OS를 짜고 싶은 1인 기업가 |
| tagline | Notion OS + 자동화 + 랜딩(선택)까지 맞춤 구축 |

**제공물**

- 인터뷰 1회 (60분) + 요구사항 정리
- 맞춤 Notion OS (DB·뷰·템플릿)
- GAS/n8n 자동화 1~2개 (범위 내)
- 인수인계 문서 + 2주간 수정 1회

**제외**

- 장기 유지보수, 풀스택 앱 전체 개발 (별도 견적)

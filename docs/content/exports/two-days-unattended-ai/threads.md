# Threads — 메인 포스트

바리스타 출신 비전공자가 AI한테 이틀을 맡기고 절반은 자리를 비웠다.

결과: 코드 변경 제안 34건, 그중 29건 반영.

비결은 좋은 AI가 아니라 규칙 3개였다.

1. 설정이 이상하면 조용히 계속 가지 말고 즉시 죽어라
2. AI는 제안까지만 — 반영 버튼은 사람이 누른다
3. 고친 AI와 검사하는 AI를 분리한다

사고도 3번 터졌다. 다만 전부 몇 분짜리 — 지난주엔 8시간짜리였다. 사고는 없애는 게 아니라 싸게 만드는 것. 그게 구조가 하는 일이더라.

무인 AI 돌려본 사람들, 어디까지 맡겨봤어?

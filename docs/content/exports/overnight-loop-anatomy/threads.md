# Threads — 메인 포스트

자기 전에 "자는 동안 돌려놔" 한 줄 남기고 잤다.

아침에 보고서가 와 있다 — 결함 27개 찾아서 4개 고쳤으니 반영할지 정하라고.

이 밤루프의 규칙은 하나다. 밤새 AI는 아무것도 스스로 반영 못 한다.

발굴 → 수정 → 다른 AI가 공격 검증 → 아침 보고. 네 단계 어디에도 "반영"이 없고, 승인 버튼은 아침에 내가 누른다.

이번 밤엔 헛돈도 있었다 — 4건 고쳤는데 알고 보니 같은 결함을 두 번씩 고친 거였다. 그래도 그걸 잡아낸 것 역시 아침 보고였고, 버리는 비용은 클릭 몇 번.

실패해도 싼 구조 위에서만 무인이 돌아간다.

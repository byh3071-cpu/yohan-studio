# 캡션

나는 바리스타다. 전공도 경력도 없다. 그런데 내가 만든 CLI가 npm에 올라가 있다. 비결은 코딩 지식이 아니라 구조였다.

AI랑 코딩하면 빠르지만, "다 됐어요"라는 말에 반복해서 데였다. 그래서 테스트를 통과 못 하면 도구가 완료 자체를 거부하게 만들었다. 한 번에 목표 하나, 모르면 "이 줄 뭐야?", 됐다는 말 대신 테스트.

전체 과정은 프로필 링크 블로그에.

# Threads — 메인 포스트

나는 바리스타다. 전공도 개발 경력도 없다. 그런데 내가 만든 CLI가 npm에 올라가 있다.

몇 달간 제일 크게 데인 건 코딩이 아니라 "다 됐어요"라는 말이었다. AI가 됐다고 해서 열어보면 버튼이 안 눌리고, 결과 파일이 안 생겨 있었다. 그거 찾느라 반나절을 날린 적도 있다.

그래서 다짐 대신 구조를 만들었다. 테스트를 통과 못 하면 도구가 "완료"를 거부하게. 그게 VHK다. 지금은 테스트 1162개가 통과해야 끝난 걸로 친다.

AI랑 일하는 사람들, "됐다"는 말 뭘로 검증하고 있어?

구조라는 게 거창한 건 아니다. 딱 세 개다.

vhk preflight — 배포 전 안전점검 일괄.

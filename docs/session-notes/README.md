# Session notes (append-only)

- One file per calendar day: `YYYY-MM-DD.md` (create when you have detail to keep out of `CLAUDE.md`).
- Append new bullets or sections at the bottom; avoid rewriting history in old entries.
- Link the active day from `CLAUDE.md` → `## 현재 상태 (노뚝이)` → **참고**.

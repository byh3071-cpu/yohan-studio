## 1. 도파민 러너 (웹게임)

- **출처 글**: [blog.naver.com/yohan3071/224144961035](https://m.blog.naver.com/yohan3071/224144961035) — "[무료 배포] 심심할 때 하는 '도파민 러너' 웹게임 출시"
- **게임 URL**: https://dopamine-runner.vercel.app/ (Vercel 배포, 설치·회원가입 불필요)

### 점검 내역
1. **페이지 로드 콘솔**: 에러 0건, 경고 0건.
2. **페이지 소스 내 시크릿 노출 점검**: `api_key`, `secret`, `token`, `sk-...`, `AIza...`(구글), `ghp_...`(깃허브) 등 패턴으로 전수 스캔 — **일치 없음**. 인라인 스크립트 1개(번들 전체 포함, 약 29KB)뿐이며 외부 API 키를 클라이언트에 하드코딩한 흔적 없음.
3. **게임 플레이 중 콘솔**: 게임 시작 → 점프 조작까지 진행하며 재확인 — 에러 0건.
4. **로컬 저장소**: `localStorage`에 `runnerCoins` 키 1개만 존재(코인 수치). 쿠키 없음. PII 저장 없음.
5. **XSS 벡터 점검**: 블로그 글에서 "점수·등급을 링크로 공유하는 기능"을 언급했으나, 실제 코드에는 `URLSearchParams`/`location.search`/`location.hash` 사용 흔적이 없어 **URL 파라미터를 페이지에 반사(reflect)하는 구조가 아님** → 반사형 XSS 벡터 없음. `innerHTML` 대입이 16곳 있으나 전부 게임 내부 상태(점수·등급 숫자)를 렌더링하는 용도로, 외부 입력이 섞이지 않음. `eval`/`new Function` 사용 없음.
6. **네트워크 요청**: 정적 리소스(폰트 CDN `cdn.jsdelivr.net`, 이미지, `site.webmanifest`)만 확인. 외부 API·분석 서버로의 의심스러운 데이터 전송 없음.

### 판정: **이상 없음.** 추가 조치 불필요.

## 2. 노션 템플릿 — "하루 수면 분석 루틴"

- **출처 글**: [blog.naver.com/yohan3071/224139003231](https://m.blog.naver.com/yohan3071/224139003231) — "뇌를 100% 활용하는 하루 수면 분석 루틴 | 노션 템플릿 무료 공유"
- **공유 링크**: `https://hyper-mallow-5b2.notion.site/2e29740ab07280f39efddef083be0ac2`

### 점검 내역
1. **접근성**: 로그인 없이 정상 열람됨(의도된 공개 공유).
2. **편집 권한**: 페이지 전체가 `disabled`(읽기 전용) 상태로 렌더링됨 — 익명 방문자가 편집 가능한 흔적 없음. 정상적인 "공유(읽기 전용)" 설정으로 보임.
3. **콘텐츠 확인**: "예시(삭제해도 됩니다)" 라벨이 붙은 샘플 데이터만 존재(예: "강의 준비", "운전 기능시험" 등 일반적인 문구). 실명·연락처·주소 등 식별 가능한 개인정보 없음.
4. **상위 페이지 노출 여부**: 사이드바·상위 워크스페이스 탐색 링크가 노출되지 않음 — 이 페이지 하나만 공개되어 있고, 비공개 워크스페이스의 다른 페이지로 새어나가는 경로 없음.
5. **콘솔 에러(30건)**: 전부 Notion 자체 내부 분석/실험 스크립트가 브라우저 CSP에 의해 차단되면서 발생하는 것(`exp.notion.com`, `cdn.metadata.io` 등)과, 익명 세션에서 `recordPageExit`/`getSubscriptionBanner` 같은 로그인 전용 내부 API가 401을 반환하는 것. **전부 Notion 플랫폼 자체의 정상적인 동작이며, 이 템플릿 소유자가 고칠 수 있는 항목이 아님.**

### 판정: **이상 없음.** 과다 공유·개인정보 노출 없음.

## 3. 노션 템플릿 — "운동 기록 DB (헬스·필라테스)"

- **출처 글**: [blog.naver.com/yohan3071/224125856258](https://m.blog.naver.com/yohan3071/224125856258) — "노션 운동 기록 템플릿 만들기 - 헬스·필라테스 자동 기록 DB (무료 공유)"
- **공유 링크**: `https://hyper-mallow-5b2.notion.site/DB-2d89740ab07280b19c14e7154117c9b2`

### 점검 내역
1. **접근성·편집 권한**: 위와 동일하게 읽기 전용으로 정상 렌더링.
2. **연결된 하위 페이지 "이용권 관리" 직접 확인**: 블로그 원문에서 저자가 "실명이나 이용권 관리에 정확한 지점 이름이 노출돼서 실제 화면은 공유 못한다"고 명시한 부분을 실제로 검증했다. 공유된 라이브 페이지의 "이용권 이름" 셀 값이 실제 헬스장/필라테스 지점명이 아니라 **컬럼 안내 문구 그대로("등록한 지점 이름")가 placeholder로 남아 있는 것**을 스크린샷으로 확인 — 저자의 설명대로 실제 개인정보(지점명)는 노출되지 않았다.
3. **콘솔 에러**: 위 항목과 동일하게 Notion 자체 내부 스크립트/CSP 이슈(플랫폼 공통 현상)로, 이 템플릿의 보안 문제가 아님.

### 판정: **이상 없음.** 저자가 우려했던 지점명 노출 없음 — 실제로 잘 마스킹되어 있다.

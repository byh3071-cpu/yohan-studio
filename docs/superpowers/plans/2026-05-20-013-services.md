# Services (013-P2) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** /services 라우트에 3개 서비스 카드 (Scan Report / OS Template / OS Build) + 외부 문의 CTA + JSON-LD `Service` 마크업을 출시한다. 결제·DB 연결 없음 (Phase 3).

**Architecture:** 정적 데이터 (`src/data/services.ts`) + 단일 페이지 (`src/app/services/page.tsx`) + 서버 컴포넌트 `ServiceCard` + `PricingSection` (값 격자). 문의 CTA는 외부 폼 (구글폼 또는 타입폼) URL — env var로 분리. JSON-LD `Service`는 페이지 내 `<script type="application/ld+json">` 인라인.

**Tech Stack:** Next.js 16 App Router · TypeScript strict · Tailwind v4 · 서버 컴포넌트만 (인터랙션 없음).

**검증 게이트:** `npm run build` + `npm run lint` + `/services` 200 + JSON-LD 파싱 (Schema.org Validator 또는 `curl + jq` 발췌).

```tsx
const SITE_ORIGIN = "https://yohan-studio.vercel.app"

function buildServiceJsonLd(contactUrl: string) {
  // Schema.org Service items @graph로 묶어 단일 script 출력
  return {
    "@context": "https://schema.org",
    "@graph": services.map((s) => ({
      "@type": "Service",
      "@id": `${SITE_ORIGIN}/services#${s.slug}`,
      name: s.name,
      description: s.tagline,
      provider: {
        "@type": "Person",
        name: "백요한",
        url: SITE_ORIGIN,
      },
      areaServed: "KR",
      url: `${SITE_ORIGIN}/services#${s.slug}`,
      offers: {
        "@type": "AggregateOffer",
        priceCurrency: "KRW",
        lowPrice: s.priceMin,
        highPrice: s.priceMax,
        availability: "https://schema.org/InStock",
        url: contactUrl || `${SITE_ORIGIN}/services`,
      },
    })),
  }
}

export default function ServicesPage() {
  const contactUrl = process.env.NEXT_PUBLIC_CONTACT_FORM_URL ?? ""
  const ld = buildServiceJsonLd(contactUrl)

  return (
    <section style={section}>
      <div style={inner}>
        <div style={head}>
          <div style={eyebrow}>{"// SERVICES — 함께 만든다"}</div>

          </p>
        </div>
        <PricingSection contactUrl={contactUrl} />
      </div>
      <script
        type="application/ld+json"
        dangerouslySetInnerHTML={{ __html: JSON.stringify(ld) }}
      />
    </section>
  )
}
```
